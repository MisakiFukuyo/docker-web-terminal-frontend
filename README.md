# docker-web-terminal-frontend

## Backend server (required)

https://github.com/TakafumiFukuyo/docker-web-terminal

## Usage

### Prepare .env file

```
HOST=0.0.0.0
PORT=9696
SERVER_PORT=9697
CHOKIDAR_USEPOLLING=true
```

### Run docker-compose up

```
docker-compose up # run nuxt.js, dev mode
```

### Open browser

http://localhost:9696
